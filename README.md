### Портфолио Frontend-разработчика MyPages

Добро пожаловать в моё портфолио! Это место - творческая мастерская, оформленная в виде монорепозитория. Здесь есть сайт-навигатор (галвная страница) и подпроекты. Всё написано на HTML, CSS, и JS с элементами TypeScript. Собрано с помощью Webpack.

--------------------------------------------------------------------

#### Сборка

Чтобы собрать проект, нужно:

- Скачать/склонировать репозиторий
- Иметь на пк установленный Node.js
- Выполнить npm install, а затем npm run build

В папке dist появится сборка, и можно будет просто открыть любую страницу

--------------------------------------------------------------------

#### Структура

- Главная страница - index.html. Представляет собой навигатор по остальным проектам. В каждом из них есть ссылка сюда.
- projects/SoftwareLanding - простой сайт из 2 страниц с лендингом программы с меню-бургером. 
- projects/RestaurantLanding - лендинг ресторана с бургером, модальными окнами и валидацией формы.
